R控制流练习题（基础）

9.1 本章学习目标

通过本章学习，你将能够：

  • 理解什么是控制流（control flow）
  • 掌握 if / ifelse / case_when 的使用
  • 理解 switch 的适用场景
  • 掌握 for 循环的基本写法
  • 理解 while 和 repeat 的区别
  • 知道什么时候应该使用循环，什么时候避免循环

9.2 什么是控制流？

在默认情况下，R 代码是按顺序执行的：

1 + 1
2 + 2
3 + 3

👉 从上到下依次执行

但在实际问题中，我们往往需要：

  • 根据条件决定是否执行某段代码
  • 重复执行某段代码
  • 对不同情况采取不同处理方式

👉 这就需要 控制流（Control Flow）
控制流是指程序在运行过程中，根据条件或规则，决定“哪些代码执行、哪些跳过、执行多少次”的机制。

9.3 条件判断（if 结构）

  • 基本语法
if (condition) {
  执行内容
}

或：

if (condition) {
  条件为真执行
} else {
  条件为假执行
}

示例：成绩等级判断

Code 
x <- 85
if (x > 90) {
  "A"
} else if (x > 80) {
  "B"
} else if (x > 50) {
  "C"
} else {
  "F"
}
[1] "B"
  • 重要特点

👉 if 只能处理 单个值（标量）

Code 
x <- c(80, 90, 70)

# if (x > 80)  ❌ 会报错或只取第一个值

9.4 向量化判断（ifelse）

当数据是向量（多个值）时，应使用：

ifelse(condition, yes, no)

示例：奇偶判断

Code 
x <- 1:10
ifelse(x %% 2 == 0, "even", "odd")
 [1] "odd"  "even" "odd"  "even" "odd"  "even" "odd"  "even" "odd"  "even"

核心理解

ifelse = 对每个元素逐个判断

👉 和 if 的区别：

结构 适用对象
if 单个值
ifelse 向量

9.5 switch 结构

👉 根据“选项”返回不同结果（类似菜单）

Code 
x <- "a"

switch(x,
  a = "option 1",
  b = "option 2",
  c = "option 3",
  stop("Invalid value")
)
[1] "option 1"
switch 主要用于“字符选择”，不适合数值比较

9.6 循环（for）

为什么需要循环？

当你需要：

重复做同一件事

例如：

  • 计算多个均值
  • 多次模拟
  • 批量处理数据
  • 基本语法
for (i in 序列) {
  执行内容
}
Code 
for (i in 1:3) {
  print(i)
}
[1] 1
[1] 2
[1] 3
  • 控制循环
Code 
for (i in 1:10) {
  
  if (i < 3) next   # 跳过本次
  
  print(i)
  
  if (i >= 5) break # 终止循环
}
[1] 3
[1] 4
[1] 5
  • 两个关键命令
语句 作用
next 跳过本次循环
break 结束整个循环

for 循环的重要原则

  • 提前分配空间（非常重要）
Code 
means <- c(1, 50, 20)

out <- vector("list", length(means))

for (i in seq_along(means)) {
  out[[i]] <- rnorm(10, means[[i]])
}

为什么？

避免在循环中不断扩展对象（效率很低）

推荐写法

seq_along(x)

比：

1:length(x)

更安全（避免长度为0时报错）

9.7 while 与 repeat

  • while 循环
while (condition) {
  执行内容
}

示例：

Code 
i <- 0

while (i < 5) {
  print("I'm happy")
  i <- i + 1
}
[1] "I'm happy"
[1] "I'm happy"
[1] "I'm happy"
[1] "I'm happy"
[1] "I'm happy"
  • repeat 循环
repeat {
  执行内容
  if (条件) break
}

示例：

Code 
i <- 0

repeat {
  print("Hi")
  i <- i + 1
  
  if (i > 4) break
}
[1] "Hi"
[1] "Hi"
[1] "Hi"
[1] "Hi"
[1] "Hi"

9.7.1 区别总结

类型 特点
while 条件控制
repeat 无限循环 + break 退出

9.8 什么时候用循环？什么时候不用？

👉 很重要的一点：

R 更推荐“向量化”，而不是循环

例如：

Code 
# 不推荐
out <- c()
for (i in 1:10) {
  out[i] <- i^2
}

# 推荐
(1:10)^2
 [1]   1   4   9  16  25  36  49  64  81 100

9.9 本章总结

  • 控制流用于控制代码执行逻辑
  • if 适用于单个条件判断
  • ifelse 适用于向量
  • case_when 适用于多条件判断
  • switch 用于选项匹配
  • for 用于重复执行
  • while 和 repeat 用于条件循环
  • 尽量使用向量化替代循环
