10  第九课 自定义函数

11 R中函数的组成

  • formals() 即函数的参数，决定怎么调用函数

  • body() 即函数体，内部的代码

  • environment() 即函数中数据对应的范围

Code 
f02 <- function(x, y) {
    x + y
}
formals(f02)
body(f02)
environment(f02)

12 R创建函数的规则

R 创建函数的规则非常简单，利用关键字function创建函数对象，然后给对象一个名字即可

Code 
f01 <- function(x) {
  sin(1 / x ^ 2)
}

13 匿名函数

有时候在调用函数时并不需要给函数名字

Code 
integrate(function(x) sin(x) ^ 2, 0, pi)
1.570796 with absolute error < 1.7e-14

14 R中的原函数

原函数(Primitive functions) 是用C语言编写的函数，可以用is.primitive()检查函数是否是 R 中的原函数

Code 
sum
function (..., na.rm = FALSE)  .Primitive("sum")

15 错误的处理办法

  • 报告错误并停止运行(stop)

  • 报告错误并继续运行(try)

15.1 



::: {.cell}

```{.r .cell-code}
f2 <- function(x) {#求负对数运算
  if(is.character(x)) stop("字符型不对数")
  if(!is.numeric(x))  stop("非数字不对数")
  if(x<0)             stop("负数不对数")
  -log(x)
}

f2(2L)
```

::: {.cell-output .cell-output-stdout}

```
[1] -0.6931472
```


:::

```{.r .cell-code}
# f2(NA)
f3 <- function(x) {#求负对数运算
   -try(log(x))
  print(paste("log 10=",log(10),sep = ""))
}
f3(-2)
```

::: {.cell-output .cell-output-stdout}

```
[1] "log 10=2.30258509299405"
```


:::

```{.r .cell-code}
f3(2)
```

::: {.cell-output .cell-output-stdout}


16.1 管道函数

Code

  deviation() %>%
  square() %>%
  mean() %>%
  sqrt()
