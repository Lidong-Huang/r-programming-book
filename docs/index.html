1 前言

本电子书为课程《R语言编程基础》课程配套电子讲义，系统介绍 R 语言的基础语法、数据处理方法、统计分析思想以及可视化技术。课程面向本科生与研究生，旨在帮助学生建立数据思维，提升科学研究与工程实践中的数据分析能力。

1.1 R语言的发展历史

R 语言起源于 20 世纪 90 年代，由新西兰奥克兰大学的 Ross Ihaka 和 Robert Gentleman 开发，最初作为统计语言 S 的开源替代版本。2000 年 R 1.0.0 正式发布，标志着其成为一个稳定的统计计算平台。

随着 CRAN（Comprehensive R Archive Network）的建立，R 逐步形成了全球性的开源生态系统。目前，R 拥有数万扩展包，涵盖：

  • 统计建模
  • 机器学习
  • 数据可视化
  • 空间分析
  • 生物信息学
  • 环境科学
  • 金融分析
  • 文献计量
  • 高性能计算

R 已成为学术界与科研领域最重要的数据分析工具之一，在统计学、生态学、农业科学、医学、社会科学等领域具有广泛应用。

1.1.1 起源阶段（1970s–1990s）：从 S 语言到 R 的诞生

  • 1970年代：S语言
    • 在 Bell Labs 由 John Chambers 等人开发
    • 目标：为统计分析提供一个交互式编程环境
    • 奠定了数据对象化和函数式编程的思想
  • 1993年：R语言诞生
    • 在 University of Auckland
    • 由 Ross Ihaka 和 Robert Gentleman 开发
    • 本质：S语言的开源实现（受 S 启发但独立开发）
    • 取名 “R” 来自两位作者名字首字母
  • 1995年：开源发布
    • 采用 GPL 许可证，推动全球协作

1.1.2 CRAN与社区形成（1997–2005）：生态系统奠基

  • 1997年：CRAN 建立
    • CRAN（Comprehensive R Archive Network）
    • 形成全球镜像网络
    • 开始支持第三方包
  • 2000年：R 1.0.0 发布
    • 标志 R 成为稳定版本
    • 基本语法和核心功能成熟
  • 统计学界广泛采用
    • 生物统计
    • 计量经济学
    • 生态学
    • 环境科学

这一阶段，R 逐渐成为统计学研究的标准工具。

1.1.3 数据科学时代（2006–2015）：R 的爆发

  • 2009年：RStudio（现 Posit）成立
    • 提供现代 IDE
    • 推动企业应用
  • Hadley Wickham 生态体系
    • 代表性包：
      • ggplot2（图形语法革命）
      • dplyr
      • tidyr
      • tidyverse
  • 2012年前后：数据科学兴起
    • R 成为数据分析主力语言之一
    • Kaggle 竞赛广泛使用 R

这一阶段，R 从“统计软件”升级为“数据科学语言”。

1.1.4 大数据与跨界融合（2016–2022）：工具链扩展

  • Shiny（交互式应用）
    • 可构建在线分析系统
  • 大数据接口与工程化
    • SparkR
    • 数据库连接（DBI）
    • 云部署支持
  • 可复现科研报告
    • R Markdown
  • 与 Python 融合
    • reticulate 包
    • 双语言协同

1.1.5 现代阶段（2023–至今）：平台化与AI融合

当前 R 呈现出几个特点：

  • 更强调可复现科学
  • 更强调数据工程能力
  • 企业级部署增强
  • 与 AI/机器学习结合（tidymodels）

代表生态：

  • tidymodels
  • Quarto（R Markdown 升级版）
  • 云端 RStudio Server
  • Docker 部署

1.2 R语言发展的核心逻辑

可以概括为四次跃迁：

阶段 核心定位
S时代 统计计算工具
CRAN时代 开源社区生态
tidyverse时代 数据科学语言
现代阶段 可复现 + 系统化科研平台

1.3 R语言在大学生编程能力培养中的重要性

在数字化与数据驱动的时代背景下，编程能力已成为大学生的重要核心能力之一。R 语言在教学中的价值主要体现在以下方面：

1.3.1 培养数据思维

R 强调数据结构、数据清洗与数据转换过程，能够帮助学生理解：

  • 数据框（data frame）结构
  • 向量化计算思想
  • 函数式编程思维
  • 可重复科研（Reproducible Research）

1.3.2 提升科研能力

R 在统计建模与可视化方面具有天然优势，能够支持：

  • 回归分析
  • 多元统计分析
  • 时间序列分析
  • 空间统计分析
  • 科学绘图与论文图件制作

对于从事科研的学生而言，R 是重要的科研工具。

1.3.3 强化跨学科能力

R 与 Python、SQL、GIS、LaTeX 等工具具有良好的兼容性，能够帮助学生构建完整的数据分析流程，提高跨学科整合能力。

通过本课程的学习，学生应能够：
