6 关于网络ppt内容的再次申名

  • 以“>”开头引导编程代码，拷贝时不要拷贝这个符号到R的工作区，或者拷贝后删掉，方可运行代码

  • “##”引导开头的是代码运行结果，请不要拷贝这个到R中

  • 有括号的变量是R内置函数，有可能有参数也有可能没有参数

10 向量索引（取子集）

R中向量的主要几种索引方式

  • 序号索引（第5位同学） 通过向量元素的位置(正整数)实现索引
Code 
x <- c(2.1, 4.2, 3.3, 5.4)#直接创建
x[1:3]
[1] 2.1 4.2 3.3
Code 
x[c(3, 1)]
[1] 3.3 2.1
Code 
x[order(x)]
[1] 2.1 3.3 4.2 5.4
Code 
# Duplicate indices will duplicate values
x[c(1, 1)]
[1] 2.1 2.1

负整数索引

Code 
x <- c(2.1, 4.2, 3.3, 5.4)#直接创建
x[-2]
[1] 2.1 3.3 5.4

空值索引

Code 
x <- c(2.1, 4.2, 3.3, 5.4)#直接创建
x[]
[1] 2.1 4.2 3.3 5.4
Code 
x[] <- 5#与x <- 5有何不同

11 逻辑索引（同学在不在）

通过TRUE与向量元素的位置对应实现索引

Code 
x <- c(2.1, 4.2, 3.3, 5.4)#直接创建
x[c(TRUE, TRUE, FALSE, FALSE)]
[1] 2.1 4.2
Code 
x[x > 3]
[1] 4.2 3.3 5.4
Code 
x[c(TRUE, FALSE)]
[1] 2.1 3.3
Code 
# Equivalent to
x[c(TRUE, FALSE, TRUE, FALSE)]
[1] 2.1 3.3

12 名称索引（张三同学）

通过预先设置名称，并按名称进行索引

Code 
x <- c(2.1, 4.2, 3.3, 5.4)#直接创建
(y <- setNames(x, letters[1:4]))
  a   b   c   d 
2.1 4.2 3.3 5.4
Code 
y[c("d", "c", "a")]
  d   c   a 
5.4 3.3 2.1
Code 
# Like integer indices, you can repeat indices
y[c("a", "a", "a")]
  a   a   a 
2.1 2.1 2.1
