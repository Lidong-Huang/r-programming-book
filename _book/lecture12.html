14 假设检验的概念

科幻故事：在人类的持续探索之下，终于在KIC8462852恒星处发现外太空文明，外星人也捕捉到人类探测器的信号和地球的坐标，2252年外星人回访地球，由于其携带大量共生生物，致使地球人发生大面积感染，且致死率极高。别无选择，人类迅速打造了一艘”诺亚方舟“，准备移居其他星球，以躲避这次灾难。方舟将尽可能多地携带健康人，以提高人类基因遗传成功的概率。就在方舟启程之刻，突然有3名无法辨别来源的同类人（或外星人）敲打舱门，要求一起逃离地球，问题：请你判断来的人是地球人还是外星人？

14.1 补充信息

地球人的身高\(h_e \sim N(\mu_e,\sigma_e)\)平均身高\(\mu_e=1.70 m,\sigma_e= 0.1 m\);外星人的信息掌握甚少。已知方舟外的3个智慧生物的平均身高\(\bar{h}=1.85 m\)。请依据以上信息回答问题（做出结论）。像这样根据部分样本的信息（\(\bar{h}\)）来推断样本属于哪一个总体的问题，属于假设检验(Hypothesis Testing)。

  • (1)提出原假设\(H_0\)和备择假设\(H_A\)
    本例中，\(H_0\) 来者是地球人；\(H_A\) 来者是外星人

  • (2)原假设正确的情况下，计算样本统计量
    本例中，\(H_0\) 来者是地球人（假设是正确的），按照这个假设 可以进一步获取地球人\(h\)的分布信息

14.4 (3)做出统计推断

计算蓝色区域的概率，以表征门外人平均身高\(\bar{h} \ge 1.85\)的概率（包括更极端的情况）

Code 
pnorm(1.85,1.7,0.1/sqrt(3))#黄色区域面积
[1] 0.9953126
Code 
1-pnorm(1.85,1.7,0.1/sqrt(3))#蓝色区域面积
[1] 0.004687384

结论：门外不可能是地球人
